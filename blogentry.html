Responsive imagery is important because the size of the screen and image could greatly impact the display of the webpage. There is a difference between resposnive webpages and tailoring the image based upon the screen size. If there is no responsive images, the load time on the phone could be extremely long. A good variable for responsive webpages is vw. View window changes the size of the image based upon the percentage of window open.
Tailoring the image is changing the image based upon exact lengths. For example when width reaches 440px, the image goes to certain width. Then when it gets to another threshold, width changes again.
These are examples of responsive width based upon the width of the window. First image uses vw and the second image uses px and media tag. Link to Article
